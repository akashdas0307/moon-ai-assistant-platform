# Moon-AI-Assistant-Platform

An autonomous AI assistant platform with hierarchical multi-agent architecture, persistent memory, and dual-interface design.

## Overview

This platform provides a desktop AI assistant with a dual-interface design: a chat interface for real-time interaction and a file workspace for collaborative work. It leverages a Python FastAPI backend and a React + Electron frontend.

## Tech Stack

- **Backend:** Python FastAPI
- **Frontend:** React + Electron
